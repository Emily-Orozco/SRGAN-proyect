# SRGAN-proyect
En base al artículo “A deep learning approach for white blood cells image generation and classification using SRGAN and VGG19”. (Ferdousi, J., Lincoln, S. I., Alom, M. K., &amp; Foysal, M., 2024) se recreó en código de python la arquitectura empleada para su SRGAN en la plataforma de Google Colab, esto usando el dataset "Blood Cells Images" obtenido de Kaggle.
